Welcome to poliBeePsync’s documentation!¶

PoliBeePsync is a simple GUI program, aimed at students of Politecnico di Milano, which purpose is to sync the files loaded by professors onto the BeeP platform.

This software is in the beta stage and the documentation is under construction.

How to Install/Upgrade/Remove¶

Installation/upgrade/removal instructions are provided in the Installation (Windows) page for Windows and Installation (Linux) for Linux. The program hasn’t been tested on Mac OS X yet.

Source code¶

Source code is available on https://github.com/jacotsu/polibeepsync. I recommend using a virtualenv in order to develop; the command used to create one may be different from the one I show below. The following instructions are for Linux.

git clone https://github.com/jacotsu/polibeepsync.git
cd polibeepsync
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
pip install -r test-requirements.txt
